# Customer Churn Prediction

## Overview

Customer churn is one of the most critical challenges faced by subscription-based and service-oriented businesses. This project uses machine learning techniques to predict whether a customer is likely to leave a company based on historical customer behavior and account information.

The project covers the complete machine learning workflow, including:

- Data Cleaning
- Exploratory Data Analysis (EDA)
- Feature Engineering
- Model Training
- Model Evaluation
- Explainable AI
- Deployment
